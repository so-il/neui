Component 组件封装支持

封装组件的交互行为，并对外提供简便的调用和触发方式。

定义Component


        Ne.component(role,config);

功能描述

组件定义方法，定义组件的模板和属性包括行为。

参数说明

参数 类型 描述
role string 组件类名称
config object 组件定义，配置项参考配置说明表。

config配置说明

项目 类型 描述
template string 组件模版，模版语法参考render说明。
props object 组件自定义属性及行为方法。

举例说明


        //封装按钮组件
        Ne.component('button', {
            template'<button class="ne-btn">{title}</button>',
            props: {
                disabled: function () {
                    this.classList.add('disabled');
                },
                enable: function () {
                    this.classList.remove('disabled');
                }
            }
        });

调用Component方法


        Ne.component.acts(selector).propsName(); 
        //或使用
        Ne.acts(selector).propsName();

功能描述

调用Component定义的方法，neui通过htmlElement的标签名或ne-role属性值来分辨element所属的组件类别。

参数说明

参数 类型 描述
selector element或string 元素对象或选择器

举例说明


        <!--假设现在有如下按钮：-->
        <button id="btn1" class="ne-btn">按钮</button>
        <a id="btn2" class="ne-btn" ne-role="button">按钮</a>
    
        //使按钮转为不可点击状态：
        Ne.acts('#btn1').disabled();
        Ne.acts('#btn2').disabled();

初始化Component


        Ne.component.init(element);

功能描述

初始化component，如果component的props中定义有init方法的话会被调用执行。

参数说明

参数 类型 描述
element element 元素对象
