Animate 动效支持

大部分时候推荐使用css完成动效，但也有可能遇到不得不使用js控制的时候，于是在此提供js控制元素完成动效的支持。

创建时间轴


        //创建
        var timeline=new Ne.motion.timeline(Configs);
        //执行
        timeline.run();
        //停止
        timeline.stop();

功能描述

创建动画过渡所需要的时间轴，针对矢量坐标进行符合某个速度曲线的数值演进计算，具体的动画在每个关键帧回调中自定义。

Configs属性说明

属性 类型 描述
origin array 向量坐标的起点。
transition array 向量坐标的终点，需要进行变化的数值集，支持多维。
duration number 时间轴长度，以毫秒为单位，默认值1000毫秒。
timingFunction string 规定动效的速率模式，支持的类型参考缓动类型表，效果演示可访问：缓动类型效果演示
keyframesLength number 关键帧数量，默认值由程序以每秒24帧的速率自动计算。
keyframeFunction functon 关键帧的执行回调，向量坐标的变化值将作为参数集传入。
complete functon 时间轴结束的执行回调。

举例说明


        //创建
        var timeline=new Ne.motion.timeline({
            transition:[0,0],
            transition:[10,20],
            duration:100,
            keyframeLength:10,
            timingFunction:'Quad-easeOut',
            keyframeFunction:function(a,b){
                console.log(a+","+b);
            }
        });
        //执行
        timeline.run();

        //执行以上demo，全长100毫秒，共10个关键帧，keyframeFunction每隔10毫秒将输出根据timingFunction计算而得的transition的演进值，transition中的10和20分别从0=>10,0=>20演进。
        //0=>10：2,4,6,7,8,9,10,10,10,10
        //0=>20：4,8,11,13,15,17,19,20,20,20

移动


        Ne.motion.move(Configs);

功能描述

触发某个元素上的某事件。

Configs属性说明

属性 类型 描述
el element 元素对象
x number x轴上的位移。
y number y轴上的位移。
duration string 动效时长，以毫秒为单位。
timingFunction string 规定动效的速率模式，支持的类型参考缓动类型表，效果演示可访问：缓动类型效果演示
callback function 动效完成后回调。

举例说明


        //移动id为el的元素向左向下分别移动500px，300毫秒完成；
         Ne.motion.move({
            el: target.querySelector('#el'),
            x: 500,
            y: 500,
            duration: 300,
            timingFunction:'Cubic-easeOut',
            callback:function(){
            }
        });
