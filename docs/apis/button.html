交互与功能

范式说明

大部分时候只需要通过html声明即可完成所有组件所有交互行为，但也会有需要通过js触发交互行为方法的情况，关于如何触发组件交互行为方法的详细介绍请查阅-《Component 组件封装支持》


        //通过Js触发组件行为功能的范式：
        Ne.acts(selector).propsName(args[]);

button支持的交互功能

方法名 说明
.active() 使按钮进入被点击状态。
.quiet() 使按钮退出被点击状态。
.disabled() 使按钮进入不可用状态。
.enable() 使按钮恢复可用。
.showloading() 使按钮进入loading状态。
.hideloading() 使按钮退出loading状态。

举例说明


        <!--假设现在有如下button：-->
        <button class="ne-btn" id="j-btn">button</button>
        <a class="ne-btn" id="j-btn" ne-role="button">button</a>
    
        //进入loading状态：
        Ne.acts('#j-btn').showloading();
        //退出loading状态：
        Ne.acts('#j-btn').hideloading();
